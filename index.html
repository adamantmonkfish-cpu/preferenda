⚙️ One-time Firebase setup required

Preferenda uses Firebase Firestore to sync votes in real time across all devices. You need a free Firebase project to get started. This takes about 3 minutes.

Step-by-step setup instructions
  1. Go to console.firebase.google.com and click Add project
  2. Name your project (e.g. my-polls), click through the steps, create it
  3. In the left sidebar click Firestore Database → Create database
  4. Choose Start in test mode (allows all reads/writes for 30 days) → Next → Enable
  5. In the left sidebar click Project settings (gear icon)
  6. Scroll to Your apps → click the </> Web icon → Register app
